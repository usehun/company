## Tis-company 프로젝트

    Tis정보기술교육센터의 웹사이트를 구현했습니다.
    HTML과 CSS를 사용하여 화면을 만들었고,  Bootstrap의 캐러셀을 사용하여 슬라이드를 만들었습니다.
    카카오 맵 API와 그누보드의 게시판 API를 사용하였고, 닷홈에서 웹호스팅을 이용하였습니다.

#### 기술
    Front-End : HTML5, CSS3, JavaScript, Bootstrap
